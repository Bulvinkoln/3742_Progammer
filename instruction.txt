Инструкция по домашнему заданию - командная работа на GitHub

1. Клонирование репозитория с GitHub по предоставленной ссылке https://github.com/Bulvinkoln/3742_Progammer.git

    - В браузере переходим по ссылке https://github.com/Bulvinkoln/3742_Progammer.git и попадаем на главную ветку проекта "Очумелые ручки" - далее "Проект".
    - Переносим Проект через кнопку "Fork" на свой аккаунт.
    - В своём аккаунте, нажав на кнопку "Code", копируем ссылку https://github.com/vvlee1974/3742_Progammer.git 
    - Открыть редактор VS Code
    - Во окне "Начало работы" в разделе "Запуск" нажать на "Клонирование репозитория Git..".
    - Вставить ссылку - https://github.com/vvlee1974/3742_Progammer.git - и выбрать (создать) папку для сохранения репозитория.
    - В Проводнике VS Code открыть созданную папку - 3742_Progammer.

2. Работа с локальным репозиторием.

    - Запускаем терминал.
    - Создаём ветку vvlee - git branch vvlee. Далее работаем только в этой ветке.
    - Добавляем любой файл - git touch instruction.txt - в папку 3742_Progammer
    - Заполняем файл instruction.txt
